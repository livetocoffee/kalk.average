Как посчитать средний балл аттестата

Средний балл аттестата - это среднее арифметическое значение от всех оценок ученика. Чтобы посчитать средний балл аттестата, нужно сложить оценки по всем предметам и разделить полученную сумму на общее количество предметов, по которым были выставлены оценки. Этот способ применим как к 9 классу, так и к 11 классу. При этом не стоит учитывать полугодовые, четвертные и другие промежуточные оценки.
Калькулятор позволяет считать средний балл аттестата отдельно для 9 или 11 класса. Данные вычислений сохраняются в памяти.

Дополнительные результаты

Статистика оценок

Показывает количество предметов, по которым выбрана оценка и общее количество предметов.
Дополнительно подсчитывается количество пятерок, четверок и троек.

Настройки калькулятора

Список предметов

Список предметов аттестата может редактироваться. Вы можете добавить/удалить предметы из общего списка, изменить порядок следования.

Предупреждения

Калькулятор правильно подсчитает средний балл аттестата только при условии, что выбраны оценки по всем предметам. Иначе будет отображаться предупреждающее сообщение.

Количество десятичных знаков

Можно изменить количество десятичных знаков после запятой. Значение по умолчанию - 2 знака, однако можно изменить на любое удобное значение от 0 до 4.

Добавить нули

Калькулятор позволяет добавить нули в десятичной части среднего балла до нужного количества.

Округление

Вариант округления: к ближайшему целому, в большую сторону, в меньшую сторону. По умолчанию установлено округление среднего балла к ближайшему целому.

Копирование результата

При нажитии на кнопку "Копирвать" или клике на цифровой итог среднего балла подробный результат вычислений копируется в буфер обмена в виде текста.
